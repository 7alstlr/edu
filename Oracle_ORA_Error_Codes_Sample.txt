IMPORTANT NOTE
--------------
The Oracle error index page contains LINKS to 25,046 individual error code pages.
However, the page itself does NOT display error names/descriptions in a single list.
Each error code requires a separate page fetch to retrieve its description.

FULL DATA COLLECTION CHALLENGE
------------------------------
To collect all 25,046 error codes with descriptions would require:
- 25,046 individual HTTP requests (one per error code page)
- Estimated time: ~2-3 hours with respectful rate limiting (50-100ms per request)
- This would be a significant load on Oracle's documentation servers

DATA CHARACTERISTICS
====================

Error Code Ranges:
- 00000-00999: Initial system errors (sessions, processes, licensing)
- 01000-01999: SQL, transaction, and redo log related errors
- 12000-12999: Network and TNS related errors
- 22000-22999: Object and LOB related errors
- 23000-23999: Replication and distributed transaction errors
- 62000-62999: Recent/specialized errors (Avro, Kafka, etc.)

OBSERVATIONS
============
1. Early error codes (00000-00999) primarily cover:
   - Session management
   - Process and resource limits
   - Memory allocation
   - Network connectivity

2. Mid-range codes (01000-01999) cover:
   - SQL syntax and compilation
   - Data constraints and integrity
   - Transaction and redo log issues
   - Database file operations

3. High-range codes (12000+) cover:
   - Network and TNS issues (12000-12999)
   - Advanced features (22000+)
   - Recently added functionality (62000+)

4. Error descriptions contain:
   - Placeholder text (indicated by 'string' or %0!s)
   - HTML entities (&#x27; for apostrophe, &quot; for quotes)
   - Parameter names and explanations

NOTES ON COMPLETE DATA COLLECTION
==================================

To collect all 25,046 error codes programmatically:
1. Extract all link URLs from https://docs.oracle.com/en/error-help/db/ora-index.html
2. Create a queue of URLs to fetch
3. Implement respectful rate limiting (50-100ms between requests)
4. Parse page titles to extract error code and description
5. Store results in pipe-delimited format

Alternative approaches:
- Use Oracle's official documentation PDF exports (if available)
- Query Oracle's database documentation API (if public access available)
- Check if there's a downloadable database of error codes

ESTIMATED COMPLETION TIME FOR FULL DATASET
===========================================
- 25,046 requests × 100ms = ~2,504 seconds = 41+ minutes
- With connection overhead: ~45-60 minutes estimated

This would require sustained network access without interruption.
