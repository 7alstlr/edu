# 일일 업무 일지 자동화 도구

Product Requirements Document (PRD)

## 1. 목표

무엇을 만드는가

매일 작성하는 일일 업무 일지를 자동으로 수집하여, 주간 보고서 형태로 작성하고 이메일로 전송하는 자동화 도구

왜 만드는가

- 일일 업무 내용을 메모로 남기는 시간 단축

- 매주 별도로 주간 보고서를 작성해야 하는 반복 업무 자동화

- 일관되고 전문적인 형식의 보고서 자동 생성

## 2. 주요 사용자

 사용자 유형  설명 
 ---  --- 
 본인  매일 일일 업무 일지를 작성하고 주간 보고서를 이메일로 받는 사용자 

## 3. 핵심 기능 목록 (MVP)

기능 1 일일 업무 일지 입력 양식

- Google Form, Typeform, Notion Forms 등을 통해 매일 업무 내용 입력

- 입력 항목 날짜, 담당자, 완료한 업무 (자유 텍스트 또는 선택지)

- 자동 타임스탠프 기록

기능 2 주간 데이터 자동 수집

- Google Sheet 또는 Notion에 입력된 일일 업무 일지를 자동으로 수집

- 매주 금요일 자동 집계 (스케줄 설정 가능)

- 사용자별로 데이터 분류

기능 3 주간 보고서 자동 생성 (이미지캡처 포함)

- 수집된 업무 내용을 정해진 템플릿 형식으로 정렬

- 일일 입력 시 첨부한 캡처사진, 스크린샷, 이미지 자동 포함

- PDF 또는 Google Doc 형식으로 자동 생성

- 보고서 헤더 작성자명, 주간 요약, 상세 업무 내역 (이미지 포함)

기능 4 Outlook 이메일 자동 전송

- 생성된 보고서를 매주 금요일 10시에 Outlook으로 자동 전송

- 받는 사람 본인 Outlook 메일

- 메일 제목 '[주간 보고서] 2024년 1월 1주차 - 홍길동'

- 첨부 파일 PDF 또는 Google Doc 링크 포함

기능 5 간단한 대시보드 (선택사항)

- 지난 보고서 목록 확인

- 다음 보고서 예정 시간 표시

- 간단한 통계 (예 이번 주 작성된 보고서 수)

## 4. 이번 버전에서 제외할 기능 (Out of Scope)

- 업무 내용 분류태그 기능 (버전 2.0에서 고려)

- 성과 평가 점수 기능

- 슬랙(Slack) 연동

- 다국어 지원

- 복잡한 권한 관리

## 5. 간단한 화면 흐름 설명

사용자 관점 일일 업무 일지 입력

- 매일 900 ~ 1800 사이에 입력 폼 링크 클릭

- 오늘 한 업무를 간단히 입력 (예 회의 자료 작성, 이메일 회신)

- 관련 캡처사진이나 이미지 파일 첨부 (선택사항)

- 제출 완료 → Google Sheet 또는 Notion에 자동 저장

백그라운드 자동화 워크플로우 (Make.com 또는 Zapier)

- 매주 금요일 1000에 트리거 발동

- Google SheetNotion에서 해당 주간 데이터 읽기

- 첨부된 이미지 파일 자동 수집

- Google Doc 템플릿 복제하여 보고서 자동 생성

- 이미지를 보고서에 자동 삽입

- 생성된 Google Doc URL을 PDF로 변환 (또는 Doc 링크 포함)

- Outlook으로 자동 발송

본인 관점 보고서 수신

- 매주 금요일 1000에 Outlook에서 주간 보고서 메일 수신

- 이미지가 포함된 전문적인 보고서 확인

- 캡처사진으로 업무 내용 한눈에 파악

## 6. 권장 기술 스택 (초보자 친화적)

 항목  도구  대체 옵션 
 ---  ---  --- 
 입력 양식  Google Forms  Typeform, Notion Forms, Microsoft Forms 
 데이터 저장  Google Sheets  Notion, Microsoft Excel Online 
 자동화 플랫폼  Make.com  Zapier, IFTTT 
 보고서 생성  Google Docs  Notion, Microsoft Word Online 
 이메일 전송  Outlook  Gmail, 회사 메일 서버 (SMTP) 

Outlook 연동 가능 확인
- ✓ Make.com Outlook 완벽 지원 (Microsoft 365 계정 연동 가능)
- ✓ Zapier Outlook 완벽 지원
- ✓ 개인 Outlook outlook.com, outlook.kr 모두 가능
- ✓ Microsoft 365 회사 계정 가능 (기관용 Outlook도 지원)

## 7. 예상 완성 시간

- 입력 양식 & SheetNotion 설정 20분

- Google Doc 보고서 템플릿 작성 (이미지 영역 포함) 40분

- Make.comZapier 자동화 워크플로우 (Outlook 연동 + 이미지 처리) 70분

- 테스트 및 조정 30분

- 합계 약 2-3시간 (초보자 기준 하루 안에 충분히 가능)

## 8. 성공 기준

- ✓ 입력 폼으로 매일 9시~18시 사이에 업무 일지 입력 가능

- ✓ 이미지캡처사진 파일 첨부 가능

- ✓ 매주 금요일 10시에 자동화 워크플로우 실행

- ✓ 이미지가 포함된 주간 보고서 자동 생성

- ✓ Outlook으로 금요일 10시에 보고서 자동 발송

- ✓ 전체 과정 (입력 → 생성 → 발송)이 수동 개입 없이 자동 실행
