# 페이즈 1-1: Google Forms 생성 가이드

일일 업무 일지를 입력받기 위한 Google Forms를 생성하는 단계입니다.

## 📋 작업 개요

**목표**: 매일 업무 일지를 입력받는 Google Forms 생성
**예상 시간**: 8분
**필수 요구사항**: Google 계정

---

## 단계별 진행

### 1단계: Google Forms 접속

1. 브라우저에서 https://forms.google.com 방문
2. Google 계정으로 로그인
3. "빈 설문지" 버튼 클릭하여 새 폼 생성

### 2단계: 폼 제목 및 설명 작성

1. "제목 없는 설문지" 클릭하여 제목 변경
   - **제목**: `일일 업무 일지`
2. 아래의 "설문지 설명 추가" 클릭하여 설명 입력
   - **설명**:
     ```
     매일의 업무 내용을 기록하는 양식입니다.
     완료한 업무, 이미지, 메모 등을 작성해주세요.
     
     작성 시간: 17:00 ~ 18:00
     주간 보고서는 매주 금요일 자동으로 생성되어 이메일로 발송됩니다.
     ```

### 3단계: 필드 추가 (총 4개 필드)

#### 필드 1: 날짜 (자동 입력) ⭐
1. "+" 버튼 클릭하여 새 질문 추가
2. **질문**: `작성 날짜`
3. **유형**: "단답형" → 드롭다운에서 "날짜" 선택
4. **필수 여부**: 필수로 설정 (토글 ON)
5. **기본값 설정** (권장):
   - 우측 슬라이더 아이콘 (⋮) 클릭
   - **"응답자의 현재 위치 기반으로 기본값 설정"** 또는 **"항상 오늘 날짜로"** 옵션 찾기
   - 아니면 **"현재 날짜로 설정"** 클릭

   **팁**: 기본값을 설정하면, 사용자가 폼을 열었을 때 이미 오늘 날짜가 선택되어 있습니다.

**완료 후**: 첫 번째 필드가 날짜로 설정되고, 기본값이 오늘 날짜

---

#### 필드 2: 담당자명
1. "+" 버튼 클릭
2. **질문**: `담당자명 (작성자)`
3. **유형**: "단답형"
4. **필수 여부**: 필수로 설정 (토글 ON)
5. **팁** (선택사항): "예: 김철수" 입력

**완료 후**: 담당자명 필드 추가됨

---

#### 필드 3: 완료한 업무 (주요 필드)
1. "+" 버튼 클릭
2. **질문**: `오늘 완료한 업무`
3. **유형**: "장문형" 선택
4. **필수 여부**: 필수로 설정 (토글 ON)
5. **팁**: "예: 회의 자료 작성, 이메일 회신, 데이터 분석" 입력
6. **도움말** (선택사항): "한 줄에 하나씩 입력해주세요."

**완료 후**: 업무 입력 필드 추가됨

---

#### 필드 4: 첨부 이미지/스크린샷
1. "+" 버튼 클릭
2. **질문**: `관련 이미지/스크린샷 (선택사항)`
3. **유형**: "파일 업로드" 선택
4. **필수 여부**: 선택사항 (토글 OFF)
5. **파일 유형 제한** (권장):
   - "파일 유형 제한" 옵션 활성화
   - 허용 유형: 이미지 (JPG, PNG, GIF 등)
6. **최대 파일 크기**: 10MB (기본값)

**완료 후**: 이미지 업로드 필드 추가됨

---

### 4단계: 폼 설정 조정

1. 상단의 설정 아이콘(⚙️) 클릭

2. **응답** 탭:
   - ✅ "응답 보기" 사용 설정
   - ✅ **"응답 수집 시 타임스탬프 추가"** 토글 ON
     - 이렇게 하면 Google Sheets에 자동으로 제출 시간이 기록됨
     - 추가 필드로 표시되지 않음 (Google Sheets에만 나타남)

3. **일반** 탭:
   - 기본값 사용 (특별한 변경 필요 없음)
   - **"제출 후 응답 보기"** 토글 ON (선택사항)
     - 사용자가 제출 후 작성한 내용을 다시 볼 수 있음

4. **프레젠테이션** 탭 (선택사항):
   - 진행 상황 표시: 사용 안 함 (OFF) 추천
   - 확인 메시지:
     ```
     감사합니다!
     작성하신 내용이 저장되었습니다.
     매주 금요일 주간 보고서가 이메일로 발송됩니다.
     ```

### 5단계: 폼 공유 및 링크 생성

1. 우상단의 "전송" 버튼 클릭
2. **공유 방법 선택**: "링크" 탭 클릭
3. **링크 생성**:
   - "링크 복사" 버튼 클릭
   - 복사된 링크 저장 (예: `https://forms.google.com/...`)

4. **링크 예시**:
   ```
   https://forms.google.com/your-form-id-here
   ```

5. **다른 공유 방법** (선택사항):
   - 이메일로 보내기
   - QR 코드 생성 및 공유
   - 웹사이트에 임베드

## ❓ FAQ: 자동 입력 방법

### Q1: 날짜를 자동으로 입력할 수 있나요?

**A**: 네, 두 가지 방법이 있습니다.

**방법 1: 기본값으로 오늘 날짜 설정 (폼에서 보임)**
- 날짜 필드 우측의 슬라이더 (⋮) 클릭
- "기본값으로 현재 날짜 설정" 또는 유사 옵션 선택
- 사용자가 폼을 열면 이미 오늘 날짜가 선택되어 있음
- 사용자는 필요시 날짜 변경 가능

**방법 2: 타임스탬프로 자동 기록 (시트에만 저장)**
- Google Forms 설정 → "응답" 탭
- "응답 수집 시 타임스탬프 추가" 토글 ON
- Google Sheets에 자동으로 제출 시간이 첫 컬럼에 기록됨
- 폼에는 표시되지 않음

**권장**: 두 방법 모두 사용하면 좋습니다:
- 필드 1 (날짜): 기본값으로 오늘 날짜 (사용자가 수정 가능)
- 타임스탬프: 제출 시간 자동 기록 (변조 방지)

---

### Q2: 시간(시:분)도 자동으로 입력할 수 있나요?

**A**: Google Forms에서 "시간" 필드 유형은 기본 기능이 제한되어 있습니다.

**해결 방법**:
1. **폼에서**: "시간" 필드는 사용자가 입력하도록 함
   - 기본값을 "현재 시간"으로 설정할 수 없음
   - 사용자가 직접 입력해야 함

2. **Sheets에서**: 타임스탬프가 자동으로 "시간:분:초"를 기록함
   - 설정 → 응답 → "타임스탬프 추가" ON
   - Google Sheets의 A컬럼에 "2026-06-11 17:30:45" 형식으로 저장

**권장**: 타임스탬프만 사용하고, 폼에서는 시간 필드를 제거합니다.

---

### Q3: 폼을 열 때마다 날짜가 자동으로 바뀌나요?

**A**: 네, 기본값을 설정하면 매번 폼을 열 때마다 "오늘" 날짜로 자동 업데이트됩니다.

예:
- 6월 11일에 폼 열기 → 기본값: 2026-06-11
- 6월 12일에 폼 열기 → 기본값: 2026-06-12 (자동 변경)

---

### Q4: 과거 날짜의 업무를 입력하고 싶으면 어떻게 하나요?

**A**: 사용자가 기본값(오늘)에서 다른 날짜로 변경할 수 있습니다.

- 날짜 필드를 클릭하면 달력이 나타남
- 원하는 날짜 선택 가능
- 과거, 미래 날짜 모두 선택 가능

---

### Q5: Google Sheets에서 자동으로 계산된 오늘 날짜를 보고 싶으면?

**A**: Google Sheets의 함수를 사용합니다.

추가 컬럼(예: F)에 다음 수식 입력:
```
=TODAY()      # 오늘 날짜 (YYYY-MM-DD 형식)
=NOW()        # 현재 날짜와 시간
```

이 함수들은 시트를 열 때마다 자동으로 업데이트됩니다.

---

### Q6: 제출 시간은 어디에 저장되나요?

**A**: 타임스탬프를 활성화하면:
- **Google Sheets의 A컬럼** (맨 왼쪽)에 자동으로 저장됨
- 형식: `2026-06-11 17:30:45`
- 이 정보는 Google Forms 편집 페이지에서는 보이지 않음 (시트에만 나타남)
