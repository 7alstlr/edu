# 페이즈 1-2: Google Sheets 응답 저장 설정 가이드

Google Forms의 응답이 자동으로 Google Sheets에 저장되도록 설정하는 단계입니다.

## 📋 작업 개요

**목표**: Google Forms 응답이 자동으로 Google Sheets에 저장되도록 연동
**예상 시간**: 7분
**필수 요구사항**: 완성된 Google Forms (페이즈 1-1)

---

## 단계별 진행

### 1단계: Google Forms에서 응답 연동 설정

1. Google Forms 편집 페이지 방문 (https://forms.google.com)
2. 페이즈 1-1에서 만든 "일일 업무 일지" 폼 선택
3. 상단 메뉴에서 **"응답"** 탭 클릭
4. 우상단의 **Google Sheets 아이콘 (표 모양)** 클릭
   - 또는 "응답 보기" → "Google Sheets 아이콘"

### 2단계: Google Sheets 생성 또는 연결

#### 옵션 A: 새로운 Google Sheets 생성 (권장)

1. "새 스프레드시트 생성" 선택
2. **스프레드시트 이름**: `일일업무일지_응답`
3. **생성** 버튼 클릭
4. 자동으로 Google Sheets가 열림

#### 옵션 B: 기존 Google Sheets에 연결

1. "기존 스프레드시트 선택" 클릭
2. 목록에서 해당 시트 선택
3. **확인** 클릭

---

### 3단계: Google Sheets 자동 형식 확인

생성된 또는 연결된 Google Sheets를 확인합니다:

#### 컬럼 구조 확인

자동으로 다음과 같은 구조로 생성됩니다:

| 컬럼 | 설명 | 예시 |
|------|------|------|
| **A** | Timestamp (타임스탬프) | 2026-06-11 17:30:45 |
| **B** | 날짜 | 2026-06-11 |
| **C** | 담당자명 (작성자) | 홍길동 |
| **D** | 오늘 완료한 업무 | 회의 자료 작성, 이메일 회신 |
| **E** | 관련 이미지/스크린샷 | https://drive.google.com/... |

#### 첫 번째 행 (헤더)

- 행 1에 자동으로 질문 제목이 헤더로 입력됨
- 필요시 헤더명을 더 짧게 수정 가능

**권장 수정**:
```
Original Header  →  Shortened Header
Timestamp       →  제출시간
날짜            →  날짜 (그대로)
담당자명         →  담당자
오늘 완료한 업무 →  업무내용
이미지          →  이미지URL
```

---

### 4단계: 데이터 저장소 설정

#### 시트 이름 변경 (선택사항)

1. 시트 탭 (하단) 우클릭
2. "시트 이름 바꾸기" 선택
3. **새로운 이름**: `응답` 또는 `raw_data`
4. 확인

#### 추가 시트 생성 (준비용)

향후 단계에서 사용할 추가 시트를 미리 생성할 수 있습니다:

1. "+" 버튼 클릭하여 새 시트 생성
2. **시트 이름**: `주간_요약` 또는 `weekly_summary`
3. 일단 빈 상태로 둠 (페이즈 3에서 사용)

---

### 5단계: 권한 및 공유 설정

#### 공유 설정 (Make.com 자동화용)

1. 우상단의 "공유" 버튼 클릭
2. **공유 설정**:
   - "링크를 알고 있는 모든 사용자가 접근 가능"하도록 설정
   - 또는 특정 이메일(make.com 계정)에만 공유

3. **권한**: "편집" 권한 부여
   - 향후 Make.com에서 시트에 데이터를 읽고 추가할 수 있도록

#### 스프레드시트 링크 저장

1. "공유" 상단의 스프레드시트 URL 복사
   - 예: `https://docs.google.com/spreadsheets/d/YOUR-SHEET-ID/edit`
2. `docs/sheet_url.txt`에 저장

## 📝 참고사항

### 자동 저장의 장점
- Google Forms 제출 → 자동으로 Google Sheets에 행 추가
- 수동으로 데이터를 복사할 필요 없음
- 데이터 일관성 보장

### 이미지 파일 저장 위치
- 첨부된 이미지는 자동으로 Google Drive에 저장됨
- Google Sheets의 E컬럼에 Google Drive 링크가 저장됨
- 이미지 원본은 `/일일 업무 일지 (응답)` 폴더에 자동 생성됨

### 데이터 백업
- Google Sheets는 자동으로 버전 관리 제공
- "도구" → "버전 기록"에서 이전 버전 복구 가능

## 🚨 주의사항

1. **컬럼 순서**: Google Forms의 질문 순서를 바꾸면 Sheets의 컬럼 순서도 변경됨
2. **필드 삭제**: 필드를 삭제한 후 새로 추가하면 새 컬럼이 추가됨 (기존 데이터 손상 주의)
3. **공유 범위**: 스프레드시트가 공개되어 있으면 누구나 읽을 수 있음 (민감한 정보 주의)
