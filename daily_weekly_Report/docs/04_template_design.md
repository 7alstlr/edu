# 페이즈 2-2: Google Docs 보고서 템플릿 작성 가이드

사용자의 Google Forms 필드를 기반으로 Google Docs에서 주간 보고서 템플릿을 만드는 단계입니다.

## 📋 작업 개요

**목표**: Make.com이 자동으로 데이터를 채워 넣을 수 있는 Google Docs 템플릿 생성
**예상 시간**: 25분
**필수 요구사항**: Google 계정

---

## 📊 Google Forms 필드 정리

템플릿에 포함될 필드:
1. 업무 진행 사항
2. 이미지
3. 업무 진행 중 발생한 이슈/장애 우려
4. 다음 주 중요 업무
5. 개선 사항

---

## 단계별 진행

### Step 1️⃣: Google Docs 생성

1. https://docs.google.com/document/create 방문
2. 또는 Google Drive → "새로만들기" → "문서"
3. 새 문서 생성됨

### Step 2️⃣: 문서 제목 설정

1. **"제목 없는 문서"** 클릭
2. **제목 입력**:
   ```
   일일 업무 일지 (Daily Work Log) - 주간 보고서 템플릿
   ```
3. Enter 클릭

### Step 3️⃣: 보고서 헤더 작성

다음 내용을 입력합니다:

```
일일 업무 일지
(Daily Work Log)

작성 기간: {{작성기간}}
담당자: {{담당자명}}
작성 날짜: {{작성날짜}}
```

**포맷팅**:
- 제목 ("일일 업무 일지"): 크기 24, **굵게**, 중앙 정렬
- 부제목 ("Daily Work Log"): 크기 14, 기울임, 중앙 정렬
- 나머지: 크기 11, 왼쪽 정렬

### Step 4️⃣: 구분선 추가

1. Enter 두 번 클릭
2. "삽입" → "가로선" 또는 "구분선"
3. 또는 수동으로 "-" 여러 개 입력

### Step 5️⃣: 첫 번째 섹션 - 업무 진행 사항

```
【 오늘의 업무 진행 사항 】

{{업무_진행_사항}}

[이미지 영역]
{{이미지_1}}
```

**포맷팅**:
- 섹션 제목: 크기 14, **굵게**, 파란색 (또는 원하는 색)
- 내용: 크기 11, 일반 텍스트
- {{}} 안의 텍스트: 이것은 Make.com이 자동으로 채워 넣을 플레이스홀더

### Step 6️⃣: 이미지 영역 추가

이미지를 삽입할 공간을 만듭니다:

```
[이미지 영역 - 폭 600px, 높이 400px]
```

또는:

1. "삽입" → "이미지"
2. 임시 이미지 삽입 (나중에 Make.com이 자동으로 바꿈)
3. 이미지 크기 조정:
   - 우클릭 → "이미지 옵션"
   - 너비: 500-600px (권장)

### Step 7️⃣: 두 번째 섹션 - 이슈/장애

```
【 이슈 및 장애 우려 】

{{이슈_장애_우려}}
```

### Step 8️⃣: 세 번째 섹션 - 다음 주 업무

```
【 다음 주 중요 업무 】

{{다음주_중요업무}}
```

### Step 9️⃣: 네 번째 섹션 - 개선 사항

```
【 개선 사항 및 메모 】

{{개선사항}}
```

### Step 1️⃣0️⃣: 최종 구조 확인

완성된 템플릿:

```
═══════════════════════════════════════

        일일 업무 일지
       (Daily Work Log)

작성 기간: {{작성기간}}
담당자: {{담당자명}}
작성 날짜: {{작성날짜}}

═══════════════════════════════════════

【 오늘의 업무 진행 사항 】

{{업무_진행_사항}}

[이미지 영역]
{{이미지_1}}

───────────────────────────────────────

【 이슈 및 장애 우려 】

{{이슈_장애_우려}}

───────────────────────────────────────

【 다음 주 중요 업무 】

{{다음주_중요업무}}

───────────────────────────────────────

【 개선 사항 및 메모 】

{{개선사항}}

═══════════════════════════════════════
```

### Step 1️⃣1️⃣: 스타일 적용 (선택사항)

보고서를 더 전문적으로 보이게 할 수 있습니다:

**색상 설정**:
- 섹션 제목: 파란색 (#1F4E78)
- 배경색: 밝은 회색 또는 무색

**폰트 설정**:
- 제목: Arial 또는 Noto Sans KR
- 본문: 같은 폰트 유지

**여백 설정**:
- "파일" → "페이지 설정"
- 위/아래: 1인치
- 좌/우: 1인치

### Step 1️⃣2️⃣: 문서 저장 및 공유

1. **Ctrl+S** 또는 자동 저장됨
2. **공유 설정**:
   - 우상단 "공유" 버튼
   - "링크를 알고 있는 모든 사용자가 편집 가능"
   - 또는 Make.com 계정에만 공유

3. **문서 URL 복사**:
   - 주소 표시줄에서 URL 복사
   - 형식: `https://docs.google.com/document/d/[ID]/edit`

## 💡 팁

### 이미지 크기 조정

Google Docs에 삽입된 이미지를 원하는 크기로 조정:
1. 이미지 클릭
2. 모서리 드래그하여 크기 조정
3. 우클릭 → "이미지 옵션" → 정확한 크기 입력

### 플레이스홀더 위치 표시

나중에 Make.com이 찾기 쉽도록:
- {{플레이스홀더}} 형식 사용
- 또는 [[ ]] 형식 사용
- 일관성 있게 사용

### 페이지 레이아웃

긴 보고서를 여러 페이지로 나누고 싶으면:
1. "삽입" → "페이지 나누기"
2. 섹션마다 페이지 구분

## 📝 참고사항

### Make.com과의 통합

페이즈 3에서 Make.com이:
1. Google Sheets에서 데이터 읽기
2. 이 Google Docs 템플릿 복제
3. 모든 {{플레이스홀더}} 자동 치환
4. 이미지 자동 삽입
5. PDF 변환 (선택사항)
6. Outlook으로 발송

따라서 **플레이스홀더가 정확해야 합니다!**

### 보안 및 권한

- 템플릿 문서는 "편집 가능"하게 공유
- Make.com 서비스 계정에 권한 부여
- 또는 "링크를 알고 있는 모든 사용자"로 설정
