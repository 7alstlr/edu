# 페이즈 1-3: 입력 양식 테스트 가이드

Google Forms와 Google Sheets가 정상적으로 작동하는지 검증하는 단계입니다.

## 📋 작업 개요

**목표**: 실제 데이터 입력이 제대로 작동하는지 검증
**예상 시간**: 5분
**필수 요구사항**: 완성된 Google Forms와 Google Sheets

---

## 단계별 진행

### 1단계: 테스트 데이터 준비

#### 테스트 케이스 4개 준비

다음과 같은 테스트 응답을 준비합니다:

**테스트 케이스 1: 기본 입력 (이미지 없음)**
```
날짜: 2026-06-09 (월요일)
담당자: 홍길동
업무: 
  - 회의 자료 작성
  - 팀 미팅 참석
  - 이메일 회신
이미지: (첨부하지 않음)
```

**테스트 케이스 2: 이미지 포함**
```
날짜: 2026-06-10 (화요일)
담당자: 홍길동
업무:
  - 데이터 분석 완료
  - 보고서 작성
이미지: 스크린샷 파일 (chart.png 또는 sample.png)
```

**테스트 케이스 3: 장문 텍스트**
```
날짜: 2026-06-11 (수요일)
담당자: 김철수
업무:
  - DB 성능 튜닝 진행
  - 쿼리 최적화 테스트
  - 팀 리뷰 미팅 (14:00-15:30)
  - 문서화 작업
이미지: (첨부하지 않음)
```

**테스트 케이스 4: 다중 이미지 포함**
```
날짜: 2026-06-12 (목요일)
담당자: 이영희
업무:
  - UI 디자인 수정
  - 프론트엔드 구현
이미지: 2-3개의 이미지 파일 업로드
```

#### 테스트용 이미지 파일 준비

테스트 케이스 2, 4에서 사용할 이미지 파일을 준비합니다:

**옵션 A: 스크린샷 사용 (권장)**
1. 데스크톱에서 Alt+Print Screen 또는 Windows+Shift+S로 스크린샷 캡처
2. 파일로 저장 (PNG 또는 JPG 형식)
3. 파일명: `test_screenshot_1.png`, `test_screenshot_2.png`

**옵션 B: 이미지 파일 다운로드**
- 구글 이미지에서 작은 이미지 다운로드
- 또는 간단한 텍스트를 포함한 이미지 직접 생성

**파일 저장 위치**: `c:\git_test\edu\daily_weekly_Report\docs\test_images\`

```powershell
# PowerShell에서 실행
mkdir "c:\git_test\edu\daily_weekly_Report\docs\test_images" -Force
```

### 2단계: 첫 번째 테스트 응답 제출

#### Google Forms에 접속

1. `docs/form_url.txt`에서 Google Forms 링크 복사
2. 새 탭에서 링크 방문
3. 또는 Google Forms 편집 페이지에서 우상단의 "눈 모양" 아이콘 클릭 (미리보기 모드)

#### 테스트 케이스 1 입력

1. **날짜**: 2026-06-09 선택
2. **담당자명**: "홍길동" 입력
3. **오늘 완료한 업무**:
   ```
   회의 자료 작성
   팀 미팅 참석
   이메일 회신
   ```
   입력
4. **이미지**: (선택사항이므로 비워두기)
5. **제출** 버튼 클릭

**예상 결과**: "감사합니다!" 메시지 표시

#### Google Sheets에서 확인

1. `docs/sheet_url.txt`에서 Google Sheets 링크 복사
2. 새 탭에서 링크 방문
3. **확인 사항**:
   - ✅ 행 2 (맨 아래)에 방금 제출한 데이터가 나타남
   - ✅ Timestamp가 자동으로 입력됨
   - ✅ 모든 필드가 올바르게 저장됨
   - ✅ 이미지 필드가 비어있음 (정상)

---

### 3단계: 두 번째 테스트 (이미지 포함)

#### Google Forms에서 테스트 케이스 2 제출

1. 동일한 구글 폼 링크 방문
2. **날짜**: 2026-06-10 선택
3. **담당자명**: "홍길동" 입력
4. **오늘 완료한 업무**:
   ```
   데이터 분석 완료
   보고서 작성
   ```
   입력
5. **이미지 업로드**:
   - "파일 선택" 또는 "폴더" 아이콘 클릭
   - `docs/test_images/test_screenshot_1.png` 선택
   - 파일이 업로드될 때까지 대기 (작은 파일이면 1-2초)
6. **제출** 버튼 클릭

#### Google Sheets에서 확인

1. 시트 새로고침 (F5)
2. **확인 사항**:
   - ✅ 행 3에 새로운 데이터 추가됨
   - ✅ 이미지 URL이 E컬럼에 나타남
   - ✅ URL은 `https://drive.google.com/...` 형식
   - ✅ 이미지 URL을 클릭하면 Google Drive에서 이미지 확인 가능

## 🔍 트러블슈팅

### 문제 1: 제출 후 Sheets에 데이터가 나타나지 않음

**원인**: Forms와 Sheets가 제대로 연동되지 않음

**해결 방법**:
1. Google Sheets 새로고침 (F5)
2. 아직도 안 나타나면:
   - Google Forms 편집 → "응답" 탭
   - "Google Sheets 아이콘" 클릭하여 다시 연결
3. 또는 구글 Forms → Sheets 연동을 다시 설정

---

### 문제 2: 이미지 업로드 불가능

**원인**: 파일 크기 초과 또는 파일 형식 오류

**해결 방법**:
1. 이미지 파일 크기 확인 (10MB 이하)
2. 파일 형식 확인 (JPG, PNG, GIF 등)
3. Google Forms 설정에서 파일 유형 제한 확인
4. 브라우저 캐시 삭제 후 다시 시도

---

### 문제 3: 한글 텍스트가 깨짐

**원인**: 인코딩 오류

**해결 방법**:
1. Google Sheets → "파일" → "스프레드시트 설정"
2. 언어를 "한국어"로 설정
3. 다시 테스트

---

### 문제 4: 이미지 URL이 저장되지 않음

**원인**: Google Drive 권한 문제

**해결 방법**:
1. Google Drive에서 "/일일 업무 일지 (응답)" 폴더 확인
2. 폴더가 없으면 Forms에서 자동 생성 대기
3. 폴더 권한이 제한되어 있으면 공개 설정 변경
