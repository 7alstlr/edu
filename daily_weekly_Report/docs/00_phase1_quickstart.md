## 📋 체크리스트

아래 3개 작업을 순서대로 완료하세요:

### ✅ 작업 1: Google Forms 생성 (8분)

**가이드**: `01_google_forms_setup.md` 참고

**빠른 요약**:
1. https://forms.google.com 방문
2. "빈 설문지" 클릭
3. 제목: `일일 업무 일지`
4. 4개 필드 추가:
   - 필드 1: 작성 날짜 (필수, 날짜 선택) **← 기본값으로 오늘 날짜 설정**
   - 필드 2: 담당자명 (필수, 단답형)
   - 필드 3: 완료한 업무 (필수, 장문형)
   - 필드 4: 이미지 (선택, 파일 업로드)
5. **⚙️ 설정** → "응답" 탭 → **"타임스탬프 추가" ON** ✅
6. 우상단 "전송" → "링크" → 링크 복사
7. 복사한 링크를 `docs/form_url.txt`에 붙여넣기

**자동 입력 설정**:
- 날짜 필드: 기본값으로 오늘 날짜 → 폼 열 때마다 자동 업데이트
- 타임스탬프: 제출 시간 자동 기록 → Google Sheets A컬럼에 저장

**결과 확인**:
```
✅ 구글 폼이 생성됨
✅ 4개 필드가 모두 추가됨
✅ form_url.txt에 링크가 저장됨
```

---

### ✅ 작업 2: Google Sheets 연동 (7분)

**가이드**: `02_google_sheets_setup.md` 참고

**빠른 요약**:
1. Google Forms 편집 페이지 → "응답" 탭
2. Google Sheets 아이콘 (표 모양) 클릭
3. "새 스프레드시트 생성" 선택
4. 이름: `일일업무일지_응답`
5. "생성" 클릭 (자동으로 Sheets 열림)
6. Sheets 링크를 복사하여 `docs/sheet_url.txt`에 붙여넣기

**결과 확인**:
```
✅ Google Sheets가 자동 생성됨
✅ 시트의 컬럼: A(시간), B(날짜), C(담당자), D(업무), E(이미지)
✅ sheet_url.txt에 링크가 저장됨
```

---

### ✅ 작업 3: 입력 테스트 (5분)

**가이드**: `03_testing_setup.md` 참고

**빠른 요약**:
1. `docs/form_url.txt`에서 링크 복사 → 방문
2. 테스트 응답 2개 입력:
   ```
   응답 1:
   - 날짜: 2026-06-09
   - 담당자: 홍길동
   - 업무: "회의 자료 작성, 이메일 회신"
   - 이미지: (비움)
   - 제출
   
   응답 2:
   - 날짜: 2026-06-10
   - 담당자: 홍길동
   - 업무: "데이터 분석, 보고서 작성"
   - 이미지: 스크린샷 업로드 (선택)
   - 제출
   ```

3. `docs/sheet_url.txt`에서 링크 복사 → 방문
4. 새로운 행 2개가 추가되었는지 확인

**결과 확인**:
```
✅ 형식 제출 후 "감사합니다" 메시지 표시
✅ Google Sheets에 2개 행이 자동 추가됨
✅ 모든 데이터가 올바르게 저장됨
✅ 이미지가 첨부되면 Google Drive 링크로 저장됨
```

## ⚡ 문제 해결

### Q: Forms과 Sheets가 연동되지 않았어요

A: 다시 한번:
1. Google Forms 편집 → "응답" 탭
2. Google Sheets 아이콘 클릭
3. "기존 스프레드시트 선택"으로 시트를 명시적으로 선택

### Q: 이미지 업로드가 안 돼요

A: 확인할 사항:
- 파일 크기 < 10MB
- 파일 형식: JPG, PNG, GIF
- 인터넷 연결 확인
- 브라우저 새로고침

### Q: 한글이 깨져요

A:
1. Google Sheets 열기
2. "파일" → "스프레드시트 설정"
3. 언어를 "한국어"로 변경
